🤠 Ranch Manager Bot

Built for RedM ranches and camps, this bot tracks cattle, manages crop orders, logs activity, and supports multi-server setups — all wrapped in a Wild West theme.

🌾 Features

  • 🐄 Herding Queues with Cooldowns
  • 🌽 Crop & Cattle Ticket System
  • 📜 Ranch Logs (Eggs, Milk, Sales)
  • 🎩 Supports Multiple Discord Servers
  • 🛠 Crafting & Frontier Economy Tools
